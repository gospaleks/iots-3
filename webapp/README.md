# webapp/ — Web application (NEW, built in a later iteration)

A fresh web app (any technology) visualizing the enhanced pipeline: the live eKuiper CEP event
feed (`sensors/events`), the Analytics enriched/predictive alerts, and the MaaS predictions
(e.g. predicted-vs-actual temp). Must be **non-blocking** — the pipeline runs whether or not
the app is up. See [docs/REQUIREMENTS-IoTS-3.md §6.4](../docs/REQUIREMENTS-IoTS-3.md) and
[docs/IoTS-3-EXPLAINED.md §7](../docs/IoTS-3-EXPLAINED.md).
